# POSIX-store

Posix store ist ein Webshop für POSIX-verwandte sachen (Linux, BSD) und einigen Extras

## Sortiment

* Linux T-shirts
* BSD T-shirts
* C T-shirt (Linux und BSD kernels sind alle in C geschrieben)
* Anti-rust T-shirts (BSD kernel toleriert Rust nicht sehr gut)
* Linux torvalds T-shirts (Leiter des Linux kernel)
* Anti-microsoft T-shirt (selbsterklärend)
* White monster (kein besonderer Grund)
